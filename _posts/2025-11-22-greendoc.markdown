GreenDoc is a Jekyll theme designed to create environmentally friendly and sustainable documentation websites. It emphasizes minimalism, efficiency, and accessibility, ensuring that your documentation is not only easy to read but also has a reduced environmental impact.

It can be used for GitHub Pages by referencing the theme in the `_config.yml` file:

remote_theme: `Open-and-Sustainable/greendoc_theme`

Further instructions and a preview can be accessed at [GreenDoc site](https://open-and-sustainable.github.io/greendoc_theme){:target="_blank"}{:rel="noopener noreferrer"}.
