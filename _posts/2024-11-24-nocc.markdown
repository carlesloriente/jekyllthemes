NOCC Jekyll Bundle. A fully featured bundle site for Jekyll.
Features a homepage, about page, tags cloud page, gallery of images page, examples post pages with comments powered by Disqus and a contact form powered by Formspree. 

Features

* Use of the latest Bootstrap 5 library
* Fully Responsive and Dual Color (light and dark)
* CSS and JS files are minimized by default
* Self-hosted Google web fonts
* Local and remote environments built-in
* Extensive use of SVG and WeBP

Plugins

* SEO optimized (uses [Jekyll SEO Tag](https://github.com/jekyll/jekyll-seo-tag))
* RSS feed (uses [Jekyll Feed](https://github.com/jekyll/jekyll-feed))
