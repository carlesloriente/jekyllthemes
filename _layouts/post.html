How to submit a theme

  1. Fork our site on GitHub
  2. Create a new post in the _posts directory and fill out the relevant YAML fields
  3. Make a 250x200 thumbnail and drop it in the thumbnails directory. List its filename in the post's markdown file.

Test it out, then push your changes up and open a pull request.
